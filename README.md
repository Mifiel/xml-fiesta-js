# XMLFiesta

Version: 0.0.0

Electronic signed document XML Protocol for Node & Browser

## Install

## Browser - Bower

```
bower install xml-fiesta --save
```

## NodeJS - NPM

```bash
npm install xml-fiesta --save
```
